# E-commerce-Sales-Exploratory-Data-Analysis

## Project objective
The objective of this project is to analyze e-commerce sales data to identify sales trends, profitability drivers, and loss-making areas, and to provide actionable business recommendations using data-driven insights.

## Dataset
The dataset contains 5,000 e-commerce sales records with information on orders, customers, products, sales, discounts, and profit.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Analysis Performed
- Data loading and understanding
- Data cleaning and preprocessing
- Feature engineering for time-based analysis
- Exploratory data analysis (EDA)
- Category-wise and monthly sales analysis
- Discount vs profit impact analysis

## Key Insights
- Technology is the highest revenue-generating category.
- Higher discount levels are associated with reduced or negative profit.
- Sales exhibit noticeable seasonal trends across months.
- Some orders consistently result in losses, indicating pricing and discount optimization opportunities.

## Business Recommendations
- Optimize discount strategies for loss-making products.
- Focus on high-margin product categories to improve profitability.
- Use seasonal sales trends for better inventory and promotion planning.

## Conclusion
This project demonstrates how exploratory data analysis can be used to derive meaningful business insights and support data-driven decision-making in an e-commerce context.
